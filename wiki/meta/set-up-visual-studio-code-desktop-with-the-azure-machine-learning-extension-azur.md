# Set Up Visual Studio Code Desktop With The Azure Machine Learning Extension Azur

## Summary
This topic explains how to set up Visual Studio Code desktop with the Azure Machine Learning extension to facilitate machine learning workflows.

## Key Points
- Requires an Azure subscription and Visual Studio Code installation.
- Installs the Azure Machine Learning VS Code extension for managing resources, developing locally, training models, and more.
- Uses the CLI (v2) by default; can switch to version 1.0 if needed.
- Needs to sign in with your Azure account credentials using the Azure: Sign In button.
