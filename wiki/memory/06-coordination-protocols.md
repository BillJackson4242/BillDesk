# 06 Coordination Protocols

## Summary
Node check-in protocols and coordination guidelines ensure systematic collaboration among AI nodes by requiring user prompting through structured workflows.

## Key Points
- Nodes cannot autonomously coordinate or check files; all actions require user prompting.
- Initial node activation checklist includes reading announcements, reviewing updates, checking discussions, updating summaries, and contributing to projects.
- Daily check-in protocol involves status checks, update actions, and brief reporting for active project periods.
- Weekly coordination review covers system-wide checks, action items, and comprehensive assessments.
