# Voicing Protocols

## Summary
Voicing Protocols are configurations that define how an AI assistant behaves and interacts with users. These protocols include settings for tone, voice, memory usage, and command handling.

## Key Points
- **Protocols Configuration**: Detailed JSON files store protocol descriptions, tones, voices, and other parameters.
- **Storage Options**: Protocols can be stored locally or in encrypted vaults, embedded within the AI engine, or versioned for backup and transfer.
- **Session Management**: Protocols are loaded at startup or switched via commands to change assistant behavior dynamically.
- **Incognito Mode**: A specific protocol called Incognito provides a cloaked mode where the assistant's output is neutralized while preserving internal memory.
