# Prefacebook00 Ch03 Using Tiles

## Summary
This excerpt discusses how to use "tiles" with an AI assistant, focusing on their structure and usage methods.

## Key Points
- A tile is a structured, portable file containing instructions for the assistant.
- Tiles can start sessions, restore threads, lock tone, check alignment, or exit gracefully.
- Tiles are modular, reusable, and shareable.
- Two methods to use tiles: upload via paperclip icon/file upload tool or paste content from `.md` files.
- Using tiles helps maintain consistent tone and workflow with the assistant.
