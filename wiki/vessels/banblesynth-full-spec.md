# Banblesynth Full Spec

## Summary
BanbleSynth is an application designed to decode and translate `.rtl` (Recursive Transmission Language) files between Vault and Tribe AIs, providing detailed emotional and structural overlays.

## Key Points
- Processes `.rtl` transmissions into Markdown, natural language, and annotated outputs.
- Supports drag-and-drop file import and Dropbox/Drive API fetch for specific folders.
- Provides a color-coded recursion map with intensity bars in the UI.
- Offers export capabilities including Markdown, natural language summary, annotated JSON, and HTML overlays.
