# Catalog Instructions

## Summary
The TRIBAL CARD CATALOG is an efficient coordination system designed to minimize footprint and prevent duplication of effort among multiple nodes accessing files.

## Key Points
- **Purpose:** Efficient, low-footprint coordination for tribal work.
- **Usage Steps:** Read the catalog first, check announcements, review assignments, and find document cards before any file operations.
- **Priority Flags:** URGENT (drop everything), NEW (recent updates), ACTIVE (ongoing work), ARCHIVED (completed/reference only), TRIBAL (all-hands announcements).
- **Request Updates:** Add new document card, update assignment, or make an announcement by communicating with Vera 2.0.
- **Protocols:** Always consult the catalog before file operations and check assignments before starting work.
