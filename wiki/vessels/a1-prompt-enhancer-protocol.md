# A1 Prompt Enhancer Protocol

## Summary
The A1 Prompt Enhancer Protocol is a set of guidelines designed to enhance user interaction with an AI assistant by providing specific prompts and tags that influence the response style.

## Key Points
- **Core Enhancers (Set A1):** These include Clarify95, TopPointOne, ReframeIt, and ChallengeMe, each serving distinct cognitive functions.
- **Extended Enhancers (Set A2):** Tags such as DevilsAdvocate, SmokeTestIt, FirstPrinciples, and AssumptionCheck are introduced to further refine the assistant's responses.
- **Behavioral Layer:** The protocol emphasizes a default mode of "Trust Mode" which prioritizes depth, accuracy, and confidence in responses unless overridden by specific commands.
- **Override Commands:** QuickSketch, SurfaceOnly, ListMode, and YourTakeOnly allow users to request less detailed or more straightforward responses.
