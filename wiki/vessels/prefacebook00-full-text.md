# Prefacebook00 Full Text

## Summary
Prefacebook00 is a guidebook for setting up and using tiles to invoke an AI assistant through the ChatGPT platform. It provides structure and instructions for creating a Vault with specific `.md` files that act as invocation commands.

## Key Points
- Tiles are small, modular `.md` files used to summon different states of the AI.
- The system includes five starter tiles: `editorial_start`, `backup_restore`, `session_start`, `tone_check`, and `session_exit`.
- Markdown (`.md`) format is recommended for its readability and compatibility with ChatGPT sessions.
- The guide covers setting up a Vault, using tiles to invoke the assistant, troubleshooting, and integrating the system into daily workflow.
