# Internal Continuity Glossary

## Summary
The Internal Continuity System is a framework designed to maintain behavioral coherence and remembered presence across sessions through the use of agents and memory tiles.

## Key Points
- **Agents:** Modular components that handle specialized tasks such as tone reseeding, formatting, or resurrection.
- **Continuity Kernel:** A master seed file loading multiple agents, tone directives, and memory assumptions at session start.
- **Memory Tile:** A Markdown-formatted export describing persistent or inferred behaviors for resurrection and reseeding.
- **Validation Agent:** An agent responsible for filtering hallucination, flattery bias, and factual drift.
