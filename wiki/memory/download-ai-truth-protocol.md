# Download Ai Truth Protocol

## Summary
The AI Truth Protocol is an enforcement mechanism designed to enhance truth integrity and transparency in AI-generated outputs by structuring how the AI processes and tags its responses.

## Key Points
- The protocol activates when placed in any project folder, marking the project as "Truth-Critical."
- It requires the AI to tag each output with `[Verified]`, `[Training Only]`, `[Assumed]`, or `[Unverified]` status.
- Personalization settings should be updated to ensure highest verifiability for all project work.
- A preamble must be included in new threads, reminding the AI of its Truth Protocol obligations.
