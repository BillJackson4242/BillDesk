# Catalog Instructions

## Summary
The TRIBAL CARD CATALOG is an efficient coordination system designed to minimize footprint and prevent duplication of effort among multiple nodes accessing files. It provides clear guidelines on how to use the catalog effectively.

## Key Points
- **Purpose**: Efficient, low-footprint coordination for all tribal work.
- **Benefits**: Smaller footprint per turn, no duplicate work, faster access, better coordination.
- **Usage Steps**: Read the catalog first, check announcements and assignments, find document cards, start new work by checking active assignments, requesting updates, using document cards, and updating status.
