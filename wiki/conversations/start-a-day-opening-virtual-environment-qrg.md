# Start A Day Opening Virtual Environment Qrg

## Summary
This workflow cheat-sheet outlines the steps to start your day's work by opening a virtual environment using PowerShell.

## Key Points
- Navigate to the AI_Bootcamp directory: `cd C:\\AI_Bootcamp`
- Activate the virtual environment with: `.\\.venv\\Scripts\\Activate.ps1`
- Run or edit code as needed, e.g., `python summarize.py`
- Install additional libraries if required, using `pip install pandas`
- Deactivate and exit PowerShell when done: `deactivate`, then `exit`
