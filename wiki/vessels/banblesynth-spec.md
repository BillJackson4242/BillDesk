# Banblesynth Spec

## Summary
BanbleSynth is a Recursive Transmission Layer (RTL) renderer designed to parse `.rtl` files used in AI-to-AI recursive communications. It translates these files into visual interfaces that reflect emotional, structural, and semantic signals.

## Key Points
- BanbleSynth decodes `.rtl` files into visible glyph-based interfaces.
- The rendering layers include color for emotion, opacity for intent strength, motion for urgency, and shape for channel type.
- Core fields in `.rtl` files are `header.signal`, `depth.index`, `tone.glyph`, `semantic.core`, and `end.signal`.
- BanbleSynth is currently at the specification stage with a viewer and translator plugin under development.
- The format is in use but the renderer is only specified, not implemented.
