# Prefacebook00 Ch04 Starter Tiles

## Summary
This topic outlines the five essential "starter tiles" designed to facilitate structured interactions with an AI assistant, specifically tailored for tasks such as documentation and session management.

## Key Points
- The five starter tiles are `editorial_start.md`, `backup_restore.md`, `session_start.md`, `tone_check.md`, and `session_exit.md`.
- Each tile serves a specific purpose: starting sessions, recovering from interruptions, verifying tone consistency, and closing sessions.
- These tiles should be stored in the `/vault/tiles/` directory for easy access.
