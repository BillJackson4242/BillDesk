# Agentupload Instructions

## Summary
This document outlines the process of adding a new agent to the Vault within the Memory Engine 2.0 system.

## Key Points
- Fill out and save the `NewAgent_Template.md` file with your agent's information, naming it descriptively (e.g., `Agent_TheAnalyst_400.md`).
- Upload the saved file to a specific folder structure: `Vault_Codex_Obsidian/_intake/kernels/`.
- The system will read and parse the agent’s behavior, log signal code and invocation method, integrate it into the active matrix, and optionally generate a reseed file.
- Optionally, upload a corresponding tile image with embedded metadata.
