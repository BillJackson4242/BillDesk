# Prefacebook00 Ch01 Invocation

## Summary
This topic introduces the concept of creating an AI assistant through specific invocation instructions stored in `.md` files, which are used to set the state and behavior of the AI. These files serve as summoning scrolls that hold intent, lock in tone, and enable session management.

## Key Points
- The system uses five small, modular `.md` files called tiles for invoking different states of the AI.
- Tiles include instructions for entering states such as Editorial, Companion, Memoryless recovery, Tone check, and Session exit.
- These tiles are essential for using other books like "Saving Capital U" and "The AI Memory Lie."
- The default format for these tiles is `.md` due to its clarity and compatibility with ChatGPT sessions.
- Users must create an offline Vault to store the five tiles and follow specific steps to begin, restore, or exit a session.
