# Agent Protocol Auditlayer Definitions

## Summary
Agent Protocol Auditlayer Definitions explain key components of the Vault system, including Agents, Protocols, and Background Audit Layers. These definitions provide insight into how these elements function within the system.

## Key Points
- **AGENTS**: Semi-autonomous entities with defined roles, triggered manually or visually, and maintained via memory.
- **PROTOCOLS**: Rule-based behavior systems that govern assistant actions without personification, residing in various forms of memory.
- **BACKGROUND AUDIT LAYER**: A permanent monitoring system reviewing assistant behaviors.
