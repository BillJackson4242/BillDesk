# Appendix A Fieldguides

## Summary
This document provides field guides for managing and maintaining ChatGPT sessions, including restarting sessions with cold starts, triggering fallback systems when necessary, and uploading agents.

## Key Points
- **FieldGuide_ColdStarts.md**: Restarting a ChatGPT session by pasting the `Vault_Reseed_Startup.md` content and confirming agent identity.
- **FieldGuide_FallbackTriggers.md**: Triggering the fallback system using specific phrases to restore default behavior or reseed with fallback files.
- **FieldGuide_AgentUpload.md**: Uploading agents through three methods: Markdown file, tile upload, or embedding in `Vault_Reseed_Startup.md`.
