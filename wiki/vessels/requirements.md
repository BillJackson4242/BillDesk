# Requirements

## Summary
This topic outlines the Python libraries required for the Memory Engine 2.0 project.

## Key Points
- The project requires `google-auth-oauthlib` version 1.1.0.
- It also needs `google-auth-httplib2` version 0.1.1.
- Additionally, `google-api-python-client` version 2.108.0 is necessary for the project.
