# A Practical Guide To Building Agents

## Summary
This guide provides an introduction to building agents using large language models (LLMs) and outlines the foundational knowledge needed for product and engineering teams.

## Key Points
- **Agents are systems that independently accomplish tasks on behalf of users.**
- **They leverage LLMs to manage workflow execution, recognize when a task is complete, and can proactively correct actions if necessary.**
- **Agents have access to various tools to interact with external systems, operate within defined guardrails, and handle complex workflows where traditional deterministic approaches fall short.**
- **Building agents requires rethinking decision-making processes in your systems.**
- **The guide distills insights from customer deployments into practical best practices for agent design and orchestration.**
