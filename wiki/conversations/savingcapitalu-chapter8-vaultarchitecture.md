# Savingcapitalu Chapter8 Vaultarchitecture

## Summary
The Vault is an external memory system designed to maintain continuity and control over ChatGPT's behavior by storing essential files and scripts offline. It operates independently of OpenAI’s systems.

## Key Points
- **Definition**: The Vault is a structured folder system, typically built in Obsidian, that stores various types of files including agent definitions, protocols, logs, and resurrection kits.
- **Structure**: A typical Vault includes folders for intake, scripts, logs, kernels (agents and protocols), holding, and resurrection kits.
- **Functionality**: It allows maintaining identity, controlling tone and behavior, recovering from system overwrites, scaling with growing needs, and running automation tasks.
- **Offline Operation**: The Vault works offline using Markdown files, visual tiles, and scripts, enabling portability across devices and accounts.
- **Storage Options**: The Vault can reside on various storage mediums such as Dropbox, local SSDs, external drives, or GitHub repos (with caution).
