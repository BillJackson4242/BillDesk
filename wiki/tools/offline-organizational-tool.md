# Offline Organizational Tool

## Summary
The topic is about finding an efficient offline organizational tool to manage and later upload notes and documents to Resurrection.

## Key Points
- **Tool Recommendation**: Obsidian is suggested as the best tool for managing complex, evolving systems due to its local-first nature and Markdown-based structure.
- **Features**:
  - Stores everything as plaintext .md files, ensuring future-proofing and portability.
  - Supports interlinking notes using wikilinks for easy cross-referencing.
  - Allows tagging and grouping through #tags, folders, backlinks, and graph views.
  - Custom frontmatter can be added to notes, such as SignalCode, ThreadID, Vessel, Tone, etc., which is beneficial for Resurrection syncing.
- **Folder Structure**: A recommended folder structure includes categories like CopyEdit, Threads, Personas, Codes, and Meta.
