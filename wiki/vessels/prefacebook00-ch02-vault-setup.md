# Prefacebook00 Ch02 Vault Setup

## Summary
PrefaceBook00 – Chapter 2: Setting Up Your Vault provides instructions on creating and maintaining a folder structure to store the files that define an assistant's behavior and memory.

## Key Points
- The Vault is a folder on your system holding files defining the assistant’s behavior and memory.
- It should be placed in a location backed up and accessible from multiple devices.
- Minimum content includes folders for tiles, sessions, exports, and logs.
- Steps to set it up involve downloading starter tiles, copying this book into the Vault, and regularly backing up the folder.
