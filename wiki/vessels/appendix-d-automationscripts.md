# Appendix D Automationscripts

## Summary
This topic covers automation scripts used within the Memory Engine 2.0 project to manage various tasks such as vault reseeding, intake monitoring, quarantine of specific content, and logging events.

## Key Points
- `vault_startup.bat`: Opens Notepad with a pre-filled reseed file for manual pasting.
- `vault_watcher.py`: Monitors an intake folder for new files and logs their detection.
- `mythic_quarantine.py`: Automatically moves files containing specific tags to a quarantine folder.
- `log_audit.py`: Logs major events or protocol changes in the system.
