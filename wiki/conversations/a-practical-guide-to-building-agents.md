# A Practical Guide To Building Agents

## Summary
This guide provides insights into building agents using large language models (LLMs), focusing on their unique capabilities and best practices.

## Key Points
- Agents are systems that independently accomplish tasks on behalf of users, leveraging LLMs to manage workflow execution and make decisions.
- They are suited for complex, multi-step workflows where traditional deterministic approaches fall short.
- Agents must have access to various tools to interact with external systems while operating within defined guardrails.
