# Savingcapitalu Chapter9 Filestrategy

## Summary
This topic outlines the file types and folder strategy used in an external memory system to ensure clarity and efficient management of files.

## Key Points
- **File Types**: `.md` (Markdown), `.docx`, `.bat`, `.py`, `.png`, and `.jpg`.
- **Purpose of Files**:
  - Markdown: Behavioral scripting, protocol definitions.
  - Word Documents: Full chapter exports, drafts.
  - Batch Scripts: Trigger reseed or file automation.
  - Python Scripts: Automation, file parsing.
  - Visual Tiles: Image-based triggers for agents and protocols.
- **Folder Strategy**: Organized into `Vault_Codex_Obsidian/`, including intake, kernels, scripts, logs, holding, resurrection kit, and visuals.
- **File Naming Best Practices**: Use hyphens or underscores, prefix with content type, avoid ambiguous names.
