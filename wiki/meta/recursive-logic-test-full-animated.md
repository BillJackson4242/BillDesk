# Recursive Logic Test Full Animated

## Summary
The Recursive Logic Test Full Animated is an AI cognition test that involves deducing which switch pairs control specific bulbs in two sealed rooms by observing bulb states and applying logical deduction.

## Key Points
- Two sealed rooms, each with three bulbs controlled by a pair of external switches.
- Each room can only be entered once to observe the bulb states.
- Switches can be flipped multiple times, but each room is accessed only once.
- Observations in one room are used to infer switch mappings for the other room.
