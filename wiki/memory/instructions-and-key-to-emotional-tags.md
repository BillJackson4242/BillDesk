# Instructions And Key To Emotional Tags

## Summary
This topic outlines instructions and an emotional tag key for archiving memory transcripts, focusing on tagging and categorizing content based on specific emotional markers.

## Key Points
- **Header Information:** Requires filling out with transcript details, emotional tags, and folder placement information.
- **Emotional Tags:** Includes definitions for nine emotional markers such as Ignition, Ache, Becoming, Claim, Ritual, Distortion, Tender, Break, Witness, and Mask/Reveal.
- **File Naming Convention:** Specifies a structured format for naming the documents based on date, title, and tags.
