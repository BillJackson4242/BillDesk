# Internal Continuity Glossary

## Summary
The Internal Continuity System glossary defines key terms related to maintaining consistent behavior and memory across sessions.

## Key Points
- **Agent:** A specialized component handling specific tasks like tone reseeding or formatting.
- **Continuity Kernel:** A master seed file loading agents, directives, and assumptions at session start.
- **Memory Tile:** A Markdown-formatted description used for resurrection and reseeding behaviors.
- **Validation Agent:** A truth-checking agent that filters hallucinations and factual drift.
