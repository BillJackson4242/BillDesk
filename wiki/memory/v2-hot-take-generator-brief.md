# V2 Hot Take Generator Brief

## Summary
The V2 Hot Take Generator is an upgraded version of Bill Jackson's social media content generator, specifically designed to produce "hot takes" based on TikTok transcripts. It uses the Anthropic API in Artifacts pattern for generating content directly from claude.ai without requiring user-supplied API keys.

## Key Points
- **Project Name:** Bill's TikTok / Instagram / LinkedIn Hot Take Generator V2.0
- **API Used:** Anthropic API in Artifacts, with `claude-sonnet-4-20250514` model
- **Deployment Environment:** Claude.ai artifact (React component)
- **Max Tokens:** 1000
- **Output Platforms:** TikTok, Instagram, LinkedIn
