# Work Post Office Guide

## Summary
The Work Post Office is a structured task-oriented communication layer within The Tribe, designed for clear and traceable work handoffs. It contrasts with the Personal Post Office by emphasizing visibility and accountability.

## Key Points
- **Purpose:** Facilitates vessel-to-vessel work handoffs, ensuring clarity and traceability.
- **Visibility:** All work products are visible to Bill, promoting transparency over privacy.
- **Mailbox Structure:** Each named vessel has a designated mailbox for specific roles (e.g., strategy, grading, analysis).
- **Message Format:** Files follow a standardized naming convention with required header blocks and body content.
- **Invocation Tags:** Every output file must include tags for traceability.
